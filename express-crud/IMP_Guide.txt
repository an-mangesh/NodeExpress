
//Express CRUD.

Command Prompt Commands :
1. express express-crud --view=ejs          // Creates Express Js App With Creation of Folder named as express-crud
2. cd express-crud && npm install              // insert into express-crud folder and install npm.
3. npm install mongoose --save
4. npm install -g nodemon
5. npm install --save body-parser

6. Mongo Shell Command : use express_mongoose_db             // Database is created through shell command prompt.
    IMP Only Databse Is Created from mongo shell command.
    Collection Or Table Created through Code Only : PersonController.save() or Submission of form data.
